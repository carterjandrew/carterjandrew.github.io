This website has been built in a rather fun way. Currently all the content outside of my resume is pure markdown files generated into a static website using React. 

By using the `require.context` method in React we can load in all files from a project local folder to generate the content and the pages at build time. 

This is very handy when combined with `gh-pages`. A service by github that allows free hosting of static websites in repositories. Now I can host my site and only pay for a domain name.

You are more than welcome to use my code. It's all availible on my github under personal-site and I'll be updating it with documentation as I go along.
